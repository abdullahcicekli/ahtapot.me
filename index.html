Powerful Features

Smart Detection

Automatically detects IPs, domains, URLs, hashes, emails, CVEs, and cryptocurrency addresses.

Built-in Search

Direct search from extension popup. No need to select text - just type and analyze instantly.

Privacy First

All API keys stored locally. No tracking, no servers, no data collection.

Lightning Fast

One-click analysis with instant results. Floating button appears automatically.

10 Intelligence Sources

Integrates with VirusTotal, OTX AlienVault, AbuseIPDB, MalwareBazaar, ARIN, Shodan, GreyNoise, URLhaus, Pulsedive, and Scamalytics for comprehensive threat intelligence.

Smart Provider Matching

See which threat intelligence providers support each indicator before analysis. No wasted API calls.

Provider Support Badges

Each detected IOC displays badge-based provider support. Know instantly which sources will analyze your data.

Secure Storage

Uses Chrome's encrypted storage for API keys with optional caching.

Modern UI

Clean design with smooth animations and tab-based results view.
